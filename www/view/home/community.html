The Ufront Community

Ask questions, get help and join in. Find others making high quality web apps with Ufront.

  • Stack Overflow (Support)

    Have a Ufront question you need answered? We're using Stack Overflow for questions and answers. Write a question and add the ufront tag so we'll see it. You can also subscribe to see all ufront questions.

  • Gitter (Chat)

    Ufront has an online chat room at Gitter. Sign in with your Github account, use web-chat, IRC, or the native apps, even request email notifications for new messages.

  • Github (Contribute)

    Bug reports, feature requests and pull requests. Github is where we discuss issues, brainstorm ideas and collaborate on code.

  • Mailing List

    We also hang out on the Haxe mailing list. It's a great place to discuss more open ended questions or hear about what's happening in the Haxe world.
