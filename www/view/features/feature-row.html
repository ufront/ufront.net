• Client and Server

    Ufront can compile your models, view and controllers to run on both the server and the client. This means your app will be fast and kick-ass for most visitors, but still work great old browsers and SEO.

  • Apps that scale

    As your app or website grows, Ufront helps you handle the complexity. With a well defined structure, separation of concerns, dependency injection and easy unit testing, Ufront projects are maintainable projects.

  • Built with Haxe

    Haxe is a superb programming language. It deserves a superb web framework. Ufront is built with the Haxe Toolkit.

  • Open source

    Ufront is an open-source, community driven project. We're backed and supported by the Haxe Foundation, and all Ufront libraries are MIT licensed.
